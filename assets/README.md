# assets/

Static image assets referenced from `README.md`.

- `banner.png` -- top hero image. Midnight mountain ridges with a single beacon relay, orange flame, sparks rising, watcher silhouette in profile. 1600 x 500.
- `triggers.png` -- horizontal row of four beacon glyphs representing the trigger set (cron / account state / price threshold / slot epoch). 1400 x 300.
- `watchman.png` -- portrait of the faceless watchman silhouette used as the ambient character. 400 x 400.

All three are produced with the fal.ai Flux Pro pipeline described in `scripts/generate-github-assets.py` (private repo). Prompts are pinned in the private direction so re-generation is reproducible.
